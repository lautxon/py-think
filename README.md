# Python Code Puzzle 🎮

**Aprende a Pensar como Programador** - Un juego educativo 2D estilo 8-bit que enseña Python y un poco más...

## Stack Tecnológico

- **React 19** - Framework UI
- **TailwindCSS v3.4.17** - Estilos
- **Lucide Icons** - Iconos
- **GSAP** - Animaciones profesionales

## Características

### 🎯 3 Niveles de Aprendizaje

1. **Nivel Fundamentos** 🌱
   - Conceptos básicos de Python
   - Variables, tipos de datos, operadores
   - 10 preguntas educativas

2. **Nivel Intermedio** 🚀
   - Estructuras de control
   - Funciones y colecciones
   - 10 preguntas desafiantes

3. **Nivel Avanzado** 💎
   - Conceptos de ingeniería de software
   - Patrones y mejores prácticas
   - 10 preguntas de nivel senior

### ✨ Experiencia Visual

- Estética 8-bit retro
- Animaciones fluidas con GSAP
- Terminal virtual interactiva
- Feedback educativo sin frustración

### 📚 Contenido Industrial

Cada nivel incluye información sobre:
- Relevancia en la industria tech
- Expectativas por nivel de experiencia
- Habilidades valoradas por empresas

## Instalación

```bash
cd pcp
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Cómo Jugar

1. **Inicio**: Haz clic en "INICIAR JUEGO"
2. **Niveles**: Lee la información industrial de cada nivel
3. **Preguntas**: Escribe 1, 2 o 3 en la terminal virtual
4. **Feedback**: Recibe explicaciones educativas
5. **Progreso**: Avanza completando los 3 niveles
6. **Resumen**: Aprende cómo piensa un programador

## Filosofía de Diseño

- ✅ **Sin frustración**: Las respuestas incorrectas muestran la solución correcta
- ✅ **Educativo**: Cada pregunta incluye explicación detallada
- ✅ **Progresivo**: Dificultad incremental fundamentos → avanzado
- ✅ **Visual**: Experiencia 8-bit inmersiva con animaciones smooth
